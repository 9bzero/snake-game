# Snake Game

Classic Snake — eat, grow, do not hit a wall or yourself.

## Features

- Keyboard (WASD / arrow keys) and touch swipe controls
- 3 difficulty levels with increasing speed
- High score saved to localStorage
- Smooth grid-based movement at 60fps
- Game-over animation and instant restart
